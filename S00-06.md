## mylog: Updating add commit push

* pull from GitHub.com (e.g. for cbkadal, "sp241")

```
cd $HOME/git/sp241/TXT/
git pull

```

[&#x213C;](#)<br id="idx05">

* Edit mylog.txt
  * Do not "echo vi", do "vi" or your other editor.
  * See also this [OS Assignment](W00-06.md).

```
echo vi mylog.txt
sleep 1
cat mylog.txt

```
